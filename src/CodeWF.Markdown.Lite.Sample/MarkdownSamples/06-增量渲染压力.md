# 06 增量渲染压力

这份文件用于在左侧编辑器里手动修改内容，观察预览区是否只刷新必要区域。可以尝试在任意段落中插入文字、删除列表项、修改代码块语言或调整表格行。

## 操作建议

1. 在本段中间插入一段中文，看预览是否快速更新。
2. 删除任务列表的一行，看后续块是否仍然保持顺序。
3. 修改代码块语言标签，例如把 `csharp` 改成 `json`。
4. 给表格增加一行，确认边框和单元格内容正常。
5. 点击“开始增量演示”，观察自动模拟会轮流执行中文段落替换、正文中部插入和文档尾部追加，每次生成的都是更接近真实中文 Markdown 文档的片段。

## 可编辑段落

当前段落用于编辑压力测试。你可以在这里插入很长的一句话，或者把其中几个词替换掉。预览刷新后，标题、后续表格、代码块和图片不应丢失。
