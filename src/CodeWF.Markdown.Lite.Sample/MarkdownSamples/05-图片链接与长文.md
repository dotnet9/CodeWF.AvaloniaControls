# 05 图片、链接与长文压力

这份文件用于检查本地图片、缺失图片 fallback、链接点击、长段落换行和特殊字符。建议在不同窗口宽度下观察预览区域是否出现不必要的横向滚动。

## 观察点

- 第一张本地图片应正常显示。
- 第二张缺失图片应显示 fallback 文本，不应导致渲染中断。
- 长链接文本应自动换行。
- 特殊字符应保持原样：`<`, `>`, `&`, `"`, `'`, `|`, `\`, `/`。

## 长段落

这是一段较长的中文说明文本，用于观察段落自动换行、行高、边距和主题颜色在实际阅读内容中的表现。MarkdownViewer 应保持文本可选择、可复制，并且在编辑区持续输入时能够增量刷新预览。切换排版主题后，这段文字的行高可能变化，但不应出现文字重叠、段落跳动过大或预览内容被裁切的问题。

This paragraph mixes English words, product names, numbers, and punctuation. It checks wrapping behavior when a line contains long identifiers like CodeWFMarkdownTableHeaderBackgroundBrush and CodeWFMarkdownInlineCodeBackgroundBrush.

## 特殊字符

| 字符 | 说明 |
| --- | --- |
| `<` 和 `>` | 普通文本里的尖括号 |
| `&` | XML/HTML 常见转义字符 |
| `|` | 表格分隔符需要放在代码中 |
| `\` | Windows 路径与转义字符 |

## 复制文本检查

选中这几段文字后使用右键菜单复制选区；在空白处使用右键菜单复制渲染文本。复制结果应包含 Markdown 的主要文本内容，而不是丢失表格、列表或代码块。
