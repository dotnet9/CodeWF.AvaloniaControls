# CodeWF.Themes

`CodeWF.Themes` contains the shared theme resources and styling assets used by the CodeWF Avalonia control suite.

## Package contents

- theme resource dictionaries
- shared brushes and styling primitives
- reusable styling support for CodeWF Avalonia controls
