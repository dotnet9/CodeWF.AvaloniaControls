# CodeWF.Core

`CodeWF.Core` provides the shared base types and infrastructure used by the CodeWF Avalonia component family.

## Package contents

- common runtime helpers
- shared abstractions used by CodeWF controls and themes
- foundational types for the CodeWF Avalonia ecosystem
