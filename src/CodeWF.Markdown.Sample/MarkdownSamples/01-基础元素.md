# 01 基础元素验收

这份文件用于快速确认常见 Markdown 元素能否正常渲染。建议切换浅色、深色和多套排版主题，观察字体大小、行高、强调色、引用背景、表格表头和代码背景是否同步变化。

## 观察点

- 标题 H1-H6 的字号应逐级递减，H2 下边框应可见。
- 段落中的 **加粗**、*斜体*、***粗斜体***、~~删除线~~、`inline code` 应保持在同一行基线附近。
- 普通链接应带下划线并可点击： [CodeWF 官网](https://codewf.com)。
- 本地图片应正常显示，点击图片应打开预览窗口。
