# CodeWF.Markdown 基础元素

这份文件覆盖常规 Markdown 元素：**加粗**、*斜体*、~~删除线~~、`行内代码`、链接和图片。图片可点击打开独立预览窗体，并支持缩放与另存为。

[CodeWF 官网](https://codewf.com) 可以作为普通链接打开。
