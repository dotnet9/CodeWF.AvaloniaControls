## C#

```csharp
public sealed record MarkdownTheme(string Name, string Key);

public static string Normalize(string? value)
{
    return string.IsNullOrWhiteSpace(value)
        ? "Simple"
        : value.Trim();
}
```

## 单元格内联元素

| 场景 | 期望 |
| --- | --- |
| `inline code` | 背景色来自固定资源 Key |
| **加粗文本** | 表格内仍保持粗体 |
| [链接](https://avaloniaui.net) | 表格内可点击 |
| 很长的说明文字 | 单元格内应自动换行，不应撑破预览宽度 |
