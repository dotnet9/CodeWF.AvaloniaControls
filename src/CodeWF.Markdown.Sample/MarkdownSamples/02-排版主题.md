# 排版主题组合预览

排版主题不只是换色，还会调整标题尺度、段落行高、引用背景和表格表头。示例应用可以把浅色/深色应用主题与每套排版主题自由组合。

## 橙心、姹紫、嫩青

这些主题更适合公众号排版和图文说明，强调色会落在标题、引用边线、链接和列表符号上。

> 主题资源通过 `MarkdownViewer` 的样式属性和 `MarkdownStyleKeys` 公开的元素类名暴露。扩展者可以新增一个 `MdThemeYourKey` 样式，并按 `MdParagraph_YourKey`、`MdCodeBlock_YourKey` 这类规则补充元素级样式。

内置排版主题清单由 `CodeWF.Markdown.Themes.MarkdownTypographyThemes` 提供。示例程序额外追加了一个 `SampleInkGreen` 排版主题，组合框中显示为“示例追加：青墨”。对应样式文件在 `Themes/SampleTypographyTheme.axaml`，可以直接复制后替换 Key 和颜色。

### 技术文档段落

构建控件库时，推荐将渲染行为与主题资源分离：控件负责解析 Markdown AST，主题包负责视觉表达。这样业务项目只需要替换或追加 Avalonia `Styles`，不需要改动渲染逻辑。

| 主题 | 适合场景 | 视觉倾向 |
| --- | --- | --- |
| 科技蓝 | 技术产品文档 | 清晰、冷静 |
| 前端之巅同款 | 前端资讯文章 | 明亮、醒目 |
| 极客黑 | 深色技术内容 | 高对比、专注 |
| 简 | 内部文档 | 克制、耐看 |

## 嵌套内容

1. 一级条目
   - 说明文字可以换行并保持缩进。
   - 继续补充细节。
2. 第二个条目
   - [x] 主题资源已加载
   - [ ] 等待自定义主题接入

```xml
<Application.Styles>
    <FluentTheme />
    <markdownThemes:MarkdownThemes />
    <StyleInclude Source="/Themes/SampleTypographyTheme.axaml" />
</Application.Styles>
```
