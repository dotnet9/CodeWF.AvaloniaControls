# CodeWF.AvaloniaControls Guide控件需求文档

## 功能列表

- Guide 控件用于实现新手引导、功能介绍、操作提示等漫游式引导场景。
- Guide 控件功能对标 AtomUI Tour 漫游式引导控件。
- 支持通过 Guide 控件开启、关闭新手引导流程。
- 支持多步骤引导，每一步可绑定一个目标控件。
- 支持无目标步骤，无目标时引导内容显示在窗口或页面中央。
- 支持当前步骤索引和总步骤数展示。
- 支持上一页、下一页、完成、关闭等基础导航操作。
- 支持在第一步隐藏或禁用上一页操作。
- 支持在最后一步显示完成操作。
- 支持用户主动关闭引导。
- 支持点击自定义操作按钮，例如跳过引导。
- 支持每个步骤配置标题。
- 支持每个步骤配置说明内容。
- 支持每个步骤配置封面内容，例如图片或自定义控件。
- 支持标题、说明、封面使用自定义展示模板。
- 支持全局步骤数据源配置，便于通过数据集合生成引导步骤。
- 支持声明式配置多个引导步骤。
- 支持每一步覆盖全局默认配置。
- 支持引导弹层显示箭头。
- 支持关闭或隐藏引导弹层箭头。
- 支持箭头指向目标控件中心点。
- 支持引导弹层根据目标控件位置自动定位。
- 支持 Center、Left、LeftTop、LeftBottom、Right、RightTop、RightBottom、Top、TopLeft、TopRight、Bottom、BottomLeft、BottomRight 等位置。
- 支持目标控件空间不足时弹层自动翻转位置。
- 支持目标控件位置变化后刷新高亮区域和弹层位置。
- 支持窗口尺寸变化、布局变化后的引导位置适配。
- 支持目标控件在滚动容器内时自动滚动到可见区域。
- 支持配置是否自动滚动到目标控件。
- 支持遮罩层显示，突出当前目标控件。
- 支持关闭遮罩层，形成非模态引导。
- 支持配置遮罩颜色。
- 支持每个步骤单独配置遮罩颜色。
- 支持目标控件高亮区域。
- 支持高亮区域与目标控件之间的水平间距。
- 支持高亮区域与目标控件之间的垂直间距。
- 支持高亮区域圆角。
- 支持高亮区域在不同 DPI、缩放和窗口状态下保持准确。
- 支持阻止遮罩模式下对背景内容的误操作。
- 支持非遮罩模式下背景内容仍可交互。
- 支持引导过程中限制目标控件滚轮滚动，避免高亮区域错位。
- 支持默认圆点进度指示器。
- 支持文本进度指示器，例如“1 / 3”。
- 支持自定义进度指示器。
- 支持进度指示器跟随当前步骤自动更新。
- 支持默认样式和强调样式。
- 支持全局样式配置。
- 支持每个步骤单独配置样式。
- 支持自定义关闭图标。
- 支持引导弹层最小宽度和最小高度适配。
- 支持引导弹层内容尺寸随内容自适应。
- 支持引导弹层动效开关。
- 支持黑色、白色和 Semi 主题下的视觉适配。
- 支持主题切换后引导控件颜色、间距、圆角等视觉元素同步刷新。
- 支持中文和英文默认文案，包括上一页、下一页、完成等按钮文本。
- 支持业务方覆盖默认文案。
- 支持动态引导场景。
- 支持在引导步骤切换时触发业务操作，例如打开菜单、展开下拉框、切换标签页、展开树节点。
- 支持在业务操作完成后定位到新出现的目标控件。
- 支持 Menu 菜单动态引导，可点击或打开菜单后圈选下拉 MenuItem。
- 支持 Popup、Flyout、ContextMenu、ComboBox 下拉项等浮层内容的动态目标定位。
- 支持目标控件延迟出现时等待并重新定位。
- 支持目标控件不存在、不可见或已卸载时跳过、重试或关闭引导。
- 支持跨容器、跨弹层、跨滚动区域的目标定位。
- 支持引导步骤开始前回调。
- 支持引导步骤完成后回调。
- 支持引导流程完成回调。
- 支持引导流程关闭回调。
- 支持自定义步骤跳转，例如直接跳到指定步骤。
- 支持重置引导状态并从第一步重新开始。
- 支持同一页面存在多个 Guide 实例，互不干扰。
- 支持空步骤时不显示引导，并保持状态稳定。
- 支持重复打开和关闭，不产生残留遮罩或弹层。
- 支持目标控件卸载、窗口关闭、页面切换时自动清理引导界面。
- 支持键盘操作，包括 Esc 关闭、方向键或 Tab 在操作按钮间导航。
- 支持焦点管理，引导打开时焦点进入引导弹层，关闭后焦点可回到触发控件。
- 支持屏幕阅读器可识别标题、说明、当前步骤和操作按钮。
- 支持提供 Guide 示例页面。
- 示例需覆盖基础引导、非模态引导、不同弹层位置、自定义指示器、自定义遮罩、自定义高亮区域、自定义操作按钮。
- 示例需覆盖动态引导，至少包含 Menu/MenuItem 下拉菜单引导场景。
